База данных
###########################################

Установщик поддерживает несколько вариантов установок базы биллинга:

:doc:`Установка СУБД на отдельном сервере<database/dedicated_server>`
=======================================================================

В данном варианте установки будет сделано:

* Установлена :abbr:`СУБД (Система управления базами данных)` - PerconaDB 8.0
* Созданы пользователи модулей
* Созданы роли пользователей
* Создана база биллинга

Данный тип установки означает что :abbr:`СУБД (Система управления базами данных)` будет находиться на отдельном сервере, запуск установщика производиться на сервере :abbr:`СУБД (Система управления базами данных)`.

:doc:`Установка на существующую СУБД<database/remote_install>`
=======================================================================

В данном варианте установки будет сделано:

* Созданы пользователи модулей
* Созданы роли пользователей
* Создана база биллинга

Данный тип установки означает что :abbr:`СУБД (Система управления базами данных)` уже установлена (MySQL/MariaDB/PerconaDB),
запуск установщика производиться на сервере :abbr:`СУБД (Система управления базами данных)` или на любом другом сервере.

:doc:`Ручная установка<database/manual_install>`
=======================================================================

В данном варианте установки все пункты установки выполняются вручную.

* Создание пользователей для модулей биллинга
* Создание ролей для пользователей
* Создание базы биллинга
* Подключение модулей биллинга к базе
